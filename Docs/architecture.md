# Architektur (Hecken-Konzept)

## Grundlagen & Dimensionen
- **Typ:** Blühende Mischhecke ("Vier-Jahreszeiten-Mix")
- **Länge:** 35 Meter
- **Höhe:** 2,0m bis 2,5m
- **Breite:** Max. 2,0m (Bewusster Kompromiss für freieren Wuchs und weniger Schnittaufwand)
- **Stückzahl:** 27 Pflanzen (Pflanzabstand ca. 1,30 Meter)

## Standort & Boden
- **Licht:** Sonnig
- **Boden:** Sandig, eher trocken ("Sieb-Boden")
- **Konsequenz:** Die Pflanzlöcher werden je nach Bedarf der Pflanzengruppe (A: "Hungrig", B: "Genügsam", C: "Wüstenkönige") spezifisch mit Kompost, Pflanzerde und Bentonit (Tonmehl) aufbereitet, um Wasser und Nährstoffe zu binden.

## Logik & Komposition
- **Winterstruktur:** Immergrüne/Halbummergrüne Pflanzen (Liguster, Feuerdorn) sind so verteilt, dass im Winter keine großen Sichtschutz-Lücken entstehen.
- **Saisonale Highlights:** Frühlingsblüher (Felsenbirne, Brautspiere), Sommerblüher (Schmetterlingsflieder, Sandrose) und Herbstfärber (Berberitze, Weigelie, Blutpflaume) lösen sich harmonisch ab.
- **Anordnung:** Ein dynamischer, asymmetrischer Pflanzrhythmus (siehe `/Docs/pflanzplan.md`) erzeugt ein natürliches Wildhecken-Bild.

## Technischer Stack & Deployment
- **Frontend:** React 19 mit Vite (Single Page Application).
- **Styling:** Modernes CSS (Glassmorphism), Lucide-Icons für die Visualisierung.
- **Hosting:** Vercel (optimiert für schnelle Auslieferung und automatische Deployments).
- **Struktur:** Der Code der Web-App befindet sich im Verzeichnis `/website`.
