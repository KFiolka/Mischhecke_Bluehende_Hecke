# Mischhecke: Blühende Hecke 2026

Eine moderne Präsentations-Website für die Planung und Visualisierung einer 35 Meter langen, ökologischen Mischhecke. Dieses Projekt dient dazu, dem Nachbarn das Konzept, die Pflanzenwahl und die zukünftige Entwicklung des Gartens anschaulich zu präsentieren.

## ✨ Features
- **Interaktiver Pflanzplan:** Detaillierte Übersicht aller 27 Sträucher mit Links zur Baumschule.
- **Wachstums-Visualisierung:** KI-generierte Bilder zeigen die Hecke nach 1, 3, 5 und 10 Jahren.
- **Die Grüne Meile:** Eine horizontale Sequenz-Map mit echten Produktfotos der Baumschule Horstmann und interaktiver Pfeil-Navigation.
- **Premium UX:** Scroll-Progress-Indikator, Floating Back-to-Top Button und sanfte Animationen.
- **Modernes UI:** Glassmorphism-Design, Lucide-Icons und ein harmonisches "Nature-Green" Farbschema.
- **Budget-Kalkulation:** Transparente Kostenaufstellung (Stand 24.04.2026) mit veredeltem Tabellen-Design.

## 🚀 Setup & Entwicklung

Das Web-Projekt befindet sich im Ordner `/website`.

```bash
# In den Website-Ordner wechseln
cd website

# Abhängigkeiten installieren
npm install

# Entwicklungs-Server starten
npm run dev

# Produktions-Build erstellen
npm run build
```
